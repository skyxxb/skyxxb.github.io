TSCTF2018

发表于 2019-04-21 | 分类于 CTF ， Web
TSCTF2018，主责web，在大佬的带领下终于进了一次线下

0x1 线上

0x01 Web

are you in class

进入题目，查看网页源代码，发现页面上的button会检查js，进入死循环

页面上有另一个button，被注释了，修改html代码后使用button提交表单，回显不在班级，在外徘徊。结合hint，在提交表单时加入X-Forwarded-For:192.168.1.3成功拿到flag。

Buy Flag

Sql的二次注入，注入点在注册界面的年龄处，将payload转换成十六进制成功绕过检查并注册成功，将payload写入数据库。之后登陆成功会调用年龄进行二次查询，此时触发sql注入。
Payload如下：

拿到Flag：TSCTF{SIMPLE_SQL_F0R_Y0U}

easy_upload

注册后使用积分购买flag，发现积分只有1000元，积分不够。

题目说是upload题目，查看sshop页面源代码，发现上传页面

官方发布hint后页面提示上传python文件

上传一个python文件，用burpsuite抓包，结合hint修改文件名（无后缀），文件类型（在content-type处修改为text/python），和文件第一行内容，成功上传

回到用户信息处发现自己的积分增加了200，重复进行文件上传，直到积分可以购买flag为止。

0x02 Misc

简单的RSA

题目给了一个bmp图片，结合题目rsa和二进制文件分析工具发现n、e的值和密文，使用在线分解n工具爆破得到p、q。

使用私钥生成工具rsatool输入p、q生成.pem文件

最后利用openssl解出明文，即flag：TSCTF{ez_rsa_real_ez~}
